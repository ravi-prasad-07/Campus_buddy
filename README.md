# Campus Buddy

Campus Buddy is a web-based campus navigation system developed to help students, faculty, and visitors easily navigate and explore the college campus. The platform provides an efficient and user-friendly solution for locating important places within the campus.

## 🚀 Features

* Campus navigation assistance
* Search for important campus locations
* User-friendly interface
* Responsive web design
* REST API integration using Flask
* Data storage using SQLite

## 🛠️ Tech Stack

### Backend

* Python
* Flask

### Frontend

* HTML
* CSS

### Database

* SQLite

## 📌 Project Objective

Large campuses can be difficult to navigate, especially for new students and visitors. Campus Buddy was developed to simplify campus navigation and improve accessibility by providing a centralized platform for finding locations easily and efficiently.
